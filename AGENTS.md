# Project Instructions

- Work 100% in English for this project: documentation, code comments, UI copy,
  tests, fixtures, commit messages, PR text, issue text, and operational notes.
- Keep source documents in their original language unless explicitly asked to
  translate them.
- Always set up Kamal for a new project, with a `.env.deploy.local` file that
  can be loaded by global `kamal`, `bin/kamal`, and `bundle exec kamal`.
